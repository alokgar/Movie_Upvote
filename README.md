# Movie_Upvote
Anyone can upvote any movie any number of times 

STEPS : 
=======for starting the server :============= 
cd Movie_upvote/
source my_env/bin/activate
cd DjangoAPI/
python manage.py runserver

======== for Angular UI : ===================== 
cd Movie_UI/movieAngularUi
ng serve 
open the browser and type url : "localhost:4200"
